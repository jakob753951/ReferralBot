# ReferralBot

Config file should have the following format:

## config.json
```json
{
    "token": "[DISCORD_BOT_TOKEN]",
    "prefix": "[Whatever prefix you want]",
    "description": "[Bot description]",
    "name": "[Bot name]",
    "referral_levels": 
    {
        "[NUMBER_LEVEL1]": ROLE_ID1,
        "[NUMBER_LEVEL2]": ROLE_ID2
    }
}
```
